# JCI Assessment Tool

**English**

JCI Assessment Tool is a lightweight, browser‑based workspace to structure, track, and review hospital compliance with JCI standards.  
This folder contains the web interface and related assets for viewing standards, recording internal assessment results, linking evidence, and monitoring follow‑up actions for quality improvement.

---

**Tiếng Việt**

JCI Assessment Tool là không gian làm việc chạy trên trình duyệt, giúp hệ thống hoá, theo dõi và rà soát mức độ tuân thủ các tiêu chuẩn JCI của bệnh viện.  
Thư mục này chứa giao diện web và các thành phần liên quan để xem tiêu chuẩn, ghi nhận kết quả tự đánh giá nội bộ, liên kết bằng chứng và theo dõi hành động khắc phục/cải tiến chất lượng.
